## 🚀 Getting Started

### Prerequisites

You only need:

* A web browser
* Git (optional)
* VS Code (recommended)

---

### Installation

#### Clone the repository

```bash
git clone https://github.com/ianuj-yadav/PORTFOLIO.git
```

#### Navigate into the project

```bash
cd PORTFOLIO
```

#### Open the project

Simply open:

```bash
index.html
```

Or run using VS Code Live Server.
